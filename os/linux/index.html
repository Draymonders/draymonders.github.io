文本处理

  • 查询socket状态
    • sudo netstat -autpn | awk '/^tcp/ { ++S[$6] } END { for(a in S) print a,S[a] }'
  • 取linux文件第4行，有用\t分割的若干个ip地址。统计出来出现次数最多的前四个。
    • 获取ip地址有两种思路
      • cat test | head -4 | tail -1 | xargs -d'\t' -r -i echo {}
      • sed -n 4p test | sed "s#\t#\n#g" (推荐)
    • 排序获取topK
      • sort -n | uniq -c | sort -r -k1 | head -4
