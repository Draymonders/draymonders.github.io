# 红利

高收益意味着高波动，低收益意味低波动。持有红利一年会比银行存款利率收益高一些，但肯定不会高很多。

看大盘指数近1年涨挺多，从10月中下旬开始，股市相对来说亏钱比赚钱容易。

红利的好处是涨涨不了多少，但跌不会比科技股跌的多

红利会分红，每份多少钱，这部分收益不体现在收益率上。

红利投的都是盈利稳定、分红持续的企业（多为成熟大型企业）。企业挣得钱一部分会回馈给股东

## 分红逻辑

- 初始：100元买净值1元的基金，持有100份。
- 分红：每份分1分（共分1元），分红后基金净值会降至0.99元。
- 现金分红：资产为 “100份×0.99元 + 1元分红”= 100元（总资产不变）。
- 红利复投：1元分红应按分红后的净值（0.99元/份） 购买，能买约 1.01 份，最终持有 101.01 份。
