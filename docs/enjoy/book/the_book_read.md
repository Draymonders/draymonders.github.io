# 读书汇总

## 理财

| 书名 | 核心观点 |
| --- | --- |
| 时势：周期波动下的国家、社会和个人 | 国家、社会与个人的变化，应放在长期周期中理解，而不是用静态视角判断趋势。<br>个人决策需要顺应宏观周期，在资产配置、职业选择与风险管理上保持节奏感。 |
| 穷爸爸，富爸爸 | 财富积累的关键，是持续持有能够产生现金流的资产，而不是不断背负负债。<br>想走向财务独立，核心不是更辛苦地工作，而是提升财商与资金使用能力。 |
| 小狗钱钱 | 理财教育越早开始越好，先建立储蓄习惯、目标意识和复利观念。<br>比追求高收益更重要的，是先形成稳定的金钱认知与执行纪律。 |
| 稀缺：我们是如何陷入贫穷与忙碌的 | 稀缺会挤占人的心智带宽，让人更容易短视、冲动，并反复陷入被动局面。<br>贫穷与忙碌不只是意志力问题，也与资源不足带来的认知负担密切相关。 |
| 定投改变命运 | 对普通人而言，长期、分散、纪律化的持续投入，比频繁择时更可行。<br>投资最重要的是建立稳定的行动制度，尽量减少情绪和波动对决策的干扰。 |
| 韭菜的自我修养 | 普通投资者首先要承认自己在信息、经验和纪律上的劣势。<br>比抓住短期机会更重要的，是仓位管理、长期主义和反人性决策能力。 |
| 财富自由之路 | 财富自由本质上是持续创造可复制价值的能力，而不是一次性的高收益。<br>认知升级、时间管理和长期积累，决定了一个人最终能拥有多少选择权。 |
| 货币未来：从金本位到区块链 | 货币形态会随着信用结构、技术条件与制度安排持续演化。<br>区块链拓展了货币与支付的技术边界，但真正决定货币能否成立的仍是信用、治理与共识。 |
| [金钱心理学：财富、人性和幸福的永恒真相](./finance/金钱心理学.md) | 财富的关键不只是收益率，更在于高储蓄率、足够耐心和长期乐观。<br>金钱最重要的意义，不是炫耀，而是换取时间自主权和内心安稳。 |
| 穷查理宝典 | 复杂问题不能依赖单一学科解释，多元思维模型能显著提升判断质量。<br>长期理性与逆向思考，比一时聪明更能帮助人避开重大错误。 |
| [纳瓦尔宝典](./finance/纳瓦尔宝典.md) | 财富、健康、幸福都重要，但真正的优先级应是健康与幸福在前。<br>长期积累、善用杠杆、把热爱做到极致，是创造财富的重要路径。 |
| [聪明的投资者](./finance/聪明的投资者.md) | 投资应以资产配置和安全边际为核心，而不是依赖短期预测。<br>真正的投资者要能控制情绪，在市场波动中保持耐心、纪律与边界感。 |
| [财富的真相](./finance/caifu_zhenxiang/index.md) | 财富来自生产效率的提升，而时间是最根本的生产资料。<br>与其沉迷投机，不如把时间投入学习、生产、销售和长期投资。 |
| 段永平投资问答录 | 投资更看重企业长期竞争力和商业模式，而不是短期价格波动。<br>做对的事、守住能力圈边界，是长期获得好结果的关键。 |

## 成长

| 书名 | 核心观点 |
| --- | --- |
| 把时间当做朋友 | 很多问题的根源并不是时间不够，而是认知方式和决策习惯存在偏差。<br>长期主义、延迟满足和持续积累，往往比技巧性的时间管理更重要。 |
| [原则](./grow/原则.md) | 直面现实、诊断问题、制定方案并持续执行，是达成目标的核心路径。<br>保持头脑开放、认真对待可信分歧，往往比坚持己见更接近正确答案。 |
| [黑天鹅：如何应对不可预知的未来](./grow/黑天鹅.md) | 世界往往由少数极端事件主导，常态化思维容易系统性误判现实。<br>与其执着于预测不可预测，不如构建能承受不确定性的结构。 |
| 被讨厌的勇气 | 人并不被过去决定，关键在于如何赋予经历意义，并主动选择当下的生活方式。<br>很多痛苦来自课题分离不清；把他人的评价还给他人，才能获得真正的自由与责任。 |
| 创业维艰 | 创业不是套用成功学模板，而是在连续坏消息中做出艰难但必要的决策。<br>CEO 最重要的职责之一，是在危机中直面问题、处理人事并保住组织生存。 |
| [人性的弱点](./grow/人性的弱点.md) | 人际关系的底层方法，是不轻易指责、真诚赞美，并站在对方角度沟通。<br>想影响别人，先要让对方感到被尊重、被理解和被重视。 |

## 传记

| 书名 | 核心观点 |
| --- | --- |
| [马斯克传](./biography/musk/index.md) | 第一性原理要求从最基本的事实出发，拆掉惯性假设后重新构建方案。<br>伟大的创新，往往来自长期使命驱动、快速迭代和高强度执行。 |
| [乔布斯传](./biography/jobs/index.md) | 真正的专注不是做更多，而是只做少数真正重要的事。<br>追求简单不是回避复杂，而是在深入理解之后把复杂问题优雅地化简。 |
| [邓小平时代](./biography/xiaoping/index.md) | 重大的改革既需要战略定力，也需要允许局部试验、逐步推广。<br>开放、务实和从实践中总结规律，是推动社会变革的重要方法。 |
| [江泽民传](./biography/jiangzemin/index.md) | 在复杂转型期，稳定大局与推动发展往往需要同时推进。<br>治理思路必须顺应时代变化，制度才能持续获得现实支撑。 |
| [黄仁勋：英伟达之芯](../year/2026.md) | 从 GPU 到 CUDA 的长期坚持，说明关键技术路线需要持续投入才能穿越周期。<br>真正拉开差距的，不只是抓住风口，而是提前为下一代计算平台做好准备。 |
| [小米创业思考](../year/2026.md) | 创业要专注、极致、快，并把口碑当作最重要的复利资产。<br>顺势而为、持续打磨产品细节，往往比一味铺摊子更重要。 |
