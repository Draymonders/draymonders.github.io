# 黑天鹅：如何应对不可预知的未来

## 核心思想

塔勒布的五个根本论断

- **世界由极端事件主导**：少数黑天鹅事件决定了历史的走向、财富的分布、科技的进程。我们以为常态是重要的，实际上异常才是。
- **人类系统性地看不到黑天鹅**：叙事谬误、确认偏误、沉默的证据、游戏谬误 —— 我们的认知系统天生就不适合处理极端不确定性。
- **我们的工具让情况更糟**：钟形曲线、正态分布、精确预测 —— 这些工具不是 "不够好"，而是在最关键的时刻给出最错误的答案。
- **制度和利益结构锁定了错误**：学术界、金融业、政治体系 —— 所有人都有动机维持虚假的确定性，没有人愿意为诚实的 "我不知道" 买单。
- **出路不在于更好的预测，而在于更好的结构**：杠铃策略、反脆弱、皮肤在场 —— 在承认不可预测的基础上构建能够抵御甚至受益于冲击的系统。


### 平均斯坦与极端斯坦

人类在一个被极端事件主导的世界里，却配备了一套只适用于温和环境的认知工具 -> 正态分布思维、线性因果推理、归纳法依赖、叙事简化 

这使我们对真正重要的黑天鹅事件系统性地视而不见。

塔勒布的目标不是让读者预测黑天鹅，而是让读者意识到自己的无知，从而在**面对不确定性时保持谦逊和韧性**。

### 人性、幸福和大笔报酬

让我提炼一下研究者所谓享乐主义幸福的主要思想。

某一年赚100万美元，但在之前9年一分钱也不赚，与在相同的时间里平均地获得相同的总收入，即10年内每年获得10万美元的收入，带来的幸福感是不同的。

反过来，第一年大赚一笔，但在余下的时间一分钱也不赚，也是一样的。出于某种原因，你的幸福系统将很快饱和，就像一笔退税带来的幸福感是不会持续很久的。

实际上，你的幸福感更多地取决于正面情绪出现的次数（心理学家称之为“积极影响”），而不是某次正面情绪的强度。

也就是说，好消息最重要的是要让人愉悦，究竟有多好并不重要。因此，要过快乐的生活，你应该尽可能平均分配这些小的“积极影响”。大量一般的好消息比一个非常好的消息更令人感到幸福。

### 智力大骗局

要把黑天鹅思想贯彻到底，你需要勇气、努力、透过现象看本质的能力以及彻底理解随机性的愿望。它还需要你不把别人的观点奉若神明。

### 随机性“美学”

高斯世界的 "美" 是虚假的秩序：

- 整齐的钟形曲线、精确的置信区间、可管理的标准差 —— **这种美是人为简化的产物，是 "平均斯坦的美学"**。

分形世界的 "美" 是真实的复杂：

- 分形图形（如曼德博集合）在无限放大中展现出无穷的细节和惊奇；
- 这种美不来自简化，而来自对复杂性的忠实呈现；
- 塔勒布认为，接受分形随机性就是接受世界本来的面目 ——**不规则、不可预测、但有其内在的结构**

"灰天鹅" 概念的正式确立：

- 黑天鹅：完全无法预见的极端事件
- 灰天鹅：虽然无法预测具体事件，但我们可以知道这类事件必然会发生，并且可以对其统计特征有所了解
- 分形 / 幂律模型不能告诉你 "明天会发生什么"，但能告诉你 **"极端事件发生的频率远比你用钟形曲线计算的要高"**
- 这就是 "灰" 的含义 ——部分可知，部分不可知
