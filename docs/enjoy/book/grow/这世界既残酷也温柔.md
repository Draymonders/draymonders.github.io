# 这世界既残酷也温柔

## 书籍摘抄

### 责任与选择

**除了你自己，没有人有义务为个人兜底，包括国家。**

因为我知道创业、冒险、竞争，这都是我自己做出的选择，我从来都没有想过再找一份稳定的工作。因此痛苦、煎熬、焦虑就是我的宿命，因此而获得进步、成长、飞跃，也是我应得的。

我认为真正的成年人只有一个标准，就是他能否自由地做出选择，是否明白自由选择的代价，并为这个自由的选择愿意承担责任。

### 竞争、价值与行动

互联网私营企业是最看重结果的领域，结果导向注定了这里最大程度上地避免了传统企业的弊病：一个人成功与否是拼爹或上级肯定的结果。只有在互联网私营企业，才能保持高度的竞争，而高度的竞争是一个平凡的年轻人，最终依靠自己的价值与能量获得自我提升、实现自我价值的最大动力。

我们必须知道，这个时代是一个注重增量，而不注重存量的时代，换句话说，时代更看重你将带来什么，而不是你曾经拥有什么。

社会并不会因为你遭受的苦难而放松对你的准则和要求，最终，还是要找到自己真正可以为社会创造价值的地方

永不放弃，努力追赶是对这个时代最好的感恩

钱不稀缺，稀缺的是奋斗
钱不稀缺，稀缺的是闯劲
钱不稀缺，稀缺的是个人

夯实自己的基础，等待机会到来，机会到来时，用强大的执行力抓住。如果机会没来，多想想自己是不是没有准备好。与其拿着放大镜找机会，不如赶紧开干，让自己变得更优秀。

### 志气与投入

贫贱不能移，孟子讲的意思是气度要大。就算睡出租屋，也不堕青云志。贫困不是对自己的伤害，妄自菲薄才是。心里有火光，就愿意向上。

当他回首往事的时候，不为模棱两可的选择而悔恨，也不为做了太多可有可无的事情而懊恼。他All in过、付出过、奋斗过，一次性把青春彻底燃烧，燃烧给一个伟大的事业，燃烧给一个波澜壮阔的事业，这种价值，才是真正人生该有的样子。别犹豫，全情投入，或者一无所有。

我们只有找到属于自己的救生筏，在潮汐起伏中顺流而下，在海浪翻滚中随波向前，逐渐奔向美丽新世界。

### 关系与内心

与爱人相处时，要明确婚姻不应损害任何一方在工作、事业和人格上的自由。结婚是为了让两个人更强大，而不是互相损害。与爱人相处的婚姻本应美好。

真正的快乐，不来自于周边人的认可，不来自于父母的认可，而来自于对自我人生的掌控，对自我成长的喜悦。一切快乐都来自于自身，这份快乐是最伟大、最高贵、最自由的。

真正的快乐不来自于周边人的认可，不来自于父母的认可而来自于对自我人生的掌控，对自我成长的喜悦

自己相处，追求内心的平和
与他人相处，追求世界的和平
