# 升官发财的经济规律

天涯神帖，整体是依托弱肉强食、二八原则、做事看结果的想法去撰写的

## 摘抄

**需求决定价值**

经济学标准

1. 在成本相同的前提下，以收获最大为最优。
2. 在收获相同的前提下，以成本最小为最优。

方法论

1. 实践是检验真理的第一标准。
2. 是否对自己有好处是检验真理的第二标准。

做专业技术人才与做管理人才的差异

| 分类 | 专业技术人才 | 管理人才 |
| --- | --- | --- |
| 对象 | 物与事 | 人与事 |
| 立足点 | 真理观、科学观；较真，追求局部最优 | 大局观、艺术观；适当弹性，关注人的感受 |
| 尺度 | 公平第一 | 效果第一、公平第二 | 
| 能力要求 | 技术专业能力第一 | 人际能力第一，专业技术第二，具备宽泛知识与能力，如财务与营销的一般知识 | 
| 成就感 | 渴望自己获得成就感 | 给组员成就感，尊重、认可、欣赏他们 | 
| 情感韧性 | 脸皮薄 | 脸皮厚，经得了表扬、批评、委屈与挫折 | 
| 人际态度 | 外方内方 | 外圆内方，追求结果，过程可变通 | 
| 事务范围 | 处理事情专一 | 处理杂事，从杂事中，人与人的交往中学习“人”的知识 |
